# Redux

Redux is a JavaScript [state](https://reactnative.dev/docs/next/state) container management library that provides a single store (single source of truth) to manage and store the entire state and data flow of an application.

# Apollo

[Apollo Client](https://www.apollographql.com/docs/react/why-apollo) is a state management library used to facilitate the interaction between the client-side (frontend) and the server-side (backend) of applications using GraphQL. Its capabilities include managing both local and remote states of the application. For example, it handles GraphQL data caching for local state management and data fetching (querying) for remote state management.
